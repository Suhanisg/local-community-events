# 🌟 Local Community Events

A modern React-based web application that allows users to explore, search, filter, and join local community events.

---

## 🚀 Features

- 🔎 Search events by title
- 🎯 Filter by event type, location, and date
- 📄 Detailed event view
- ✅ RSVP / Join event functionality
- ➕ Create new event form (with basic validation)
- 📊 Pagination support
- 📱 Responsive grid layout
- 🧭 Clean navigation bar
- 🎨 Modern UI with custom styling

---

## 🛠 Tech Stack

- React (Vite)
- React Router
- JavaScript (ES6+)
- CSS (Custom styling)
- Git & GitHub

## ⚙️ Installation & Setup

1. Clone the repository:
git clone https://github.com/suhanisg/local-community-events.git

2. Navigate to the project folder:
cd local-community-events

3. Install dependencies:
npm install

4. Start the development server:
npm run dev

5. Open in browser:
http://localhost:5173
